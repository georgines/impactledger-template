# Onde foi usado IA

[Início](../README.md)

---

A IA foi usada de verdade neste projeto. Aqui está o que ela fez, e o que ficou com a equipe.

## Ferramenta utilizada

| Ferramenta | Empresa | Uso no projeto |
|---|---|---|
| Claude Code | Anthropic | Desenvolvimento assistido durante todo o projeto |

## O que foi feito com IA

| Área | O que a IA fez |
|---|---|
| Contratos Solidity | Sugestões de padrões de segurança (CEI, custom errors) revisadas pela equipe |
| Frontend | Apoio na escrita de trechos de código TypeScript/React |
| Testes | Sugestão de casos de borda e lacunas de cobertura |

## O que não foi feito com IA

| Área | Responsável |
|---|---|
| Regras de negócio e arquitetura | Equipe |
| Escrita e estrutura dos testes | Equipe |
| Deploy e configuração de rede (Sepolia) | Equipe |
| Decisões de produto e escopo | Equipe |
| Documentação do projeto | Equipe |
| Auditoria de segurança | Equipe |

## Como foi usado

A IA foi usada pontualmente como ferramenta de consulta e sugestão. Todo código gerado foi revisado, ajustado e aprovado pela equipe antes de ser integrado. Nenhuma decisão de produto, negócio ou segurança foi delegada à IA.

O fluxo de trabalho seguiu TDD obrigatório: testes escritos antes do código, com validação da equipe em cada etapa.
